How to Export as PNG (1024x500):

  1. Option 1 - Screenshot (Easiest):
    • Right-click the graphic above and select "Open Image in New Tab"
    • Press F11 for fullscreen (or zoom to 100%)
    • Take a screenshot of just the graphic (use Snipping Tool on Windows, Shift+Cmd+4 on Mac, or gnome-screenshot on Linux)
    • Save as PNG
  2. Option 2 - Browser DevTools:
    • Right-click the SVG and "Inspect Element"
    • Right-click the <svg> tag in DevTools
    • Select "Capture node screenshot"
  3. Option 3 - Online Converter:
    • Save this HTML file
    • Go to cloudconvert.com or similar
    • Upload and convert to PNG at 1024x500
